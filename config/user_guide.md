# Руководство пользователя CallStorm

## Введение

CallStorm - это система для проведения массовой атаки звонками. Отличается от обычных методов умной системой входящих вызовов, делая больше ударов по жертве.

## Установка

1. Убедитесь, что у вас установлен Python 3.8 или выше
2. Скачайте репозиторий CallStorm
3. Установите зависимости:
   ```
   pip install -r requirements.txt
   ```

## Настройка

Перед использованием необходимо настроить конфигурацию в файле `config/settings.json`:

1. Добавьте целевые номера телефонов в массив `target_numbers`
2. Установите частоту звонков в параметре `call_frequency` (в секундах)
3. Установите продолжительность атаки в параметре `duration` (количество звонков)
4. Добавьте учетные данные Twilio:
   - `twilio_sid` - SID вашего Twilio аккаунта
   - `twilio_token` - токен вашего Twilio аккаунта
   - `twilio_phone` - номер телефона Twilio для совершения звонков

## Запуск приложения

Для запуска приложения выполните команду:
```
python src/main.py
```

## Использование

1. Запустите приложение
2. Выберите пункт "Запустить атаку звонками" в меню
3. Система начнет совершать звонки на указанные номера с заданной частотой
4. Для остановки атаки выберите пункт "Остановить атаку"

## Важное предупреждение

Этот инструмент предназначен только для образовательных целей и этичного тестирования. Использование для незаконных действий преследуется по закону. Убедитесь, что у вас есть разрешение на тестирование целевых систем.

## Системные требования

- Операционная система: Windows 10/11, Linux, macOS
- Python 3.8 или выше
- Аккаунт Twilio с пополненным балансом
